#Kubernetes commands

Resource usage in pods for a namespace - kubectl top pod -n splunk-system 

Resource usage in nodes  - kubectl top node 

Resource allocation in nodes - kubectl describe node -A 

 
 

Pod details - kubectl get pods -A -v 6 

kubectl get pods  -n iot-hub 


Extra pod details - kubectl get pods -o wide -n iot-hub 

 
kubectl get pods --all-namespaces 

Starting Kubernetes pods - kubectl scale deploy fci-solution --replicas=1 
 

Stopping kubernetes pods - kubectl scale deploy fci-solution --replicas=0
